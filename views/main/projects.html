OSGeo Project

Project dari OSGeo adalah sebagai berikut

Deegree

Deegree adalah salah satu geospatial software yang mengimplemenetasikan standar dari OGC (Open Geospatial Consortium) Web Service seperti WMS dan WFS.

Geomajas

Geomajas adalah Open source GIS framework yang terintegrasi ke dalam Server. Fokus Geomajas adalah menyediakan platform untuk Server-side Integration pada data geospatial.

Geoserver

Geoserver adalah Software Opensource berbasis Java yang mengijinkan user untuk men-share dan meng-edit data geospatial. Geoserver telah mengimplmentasikan standar Open Geospatial Consortium (OGC), Web Feature Service (WFS),Web Coverage Service (WCS) dan Web Map Service (WMS).
